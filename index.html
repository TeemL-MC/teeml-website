RECRUITMENT

Let us do the hard work for you. We can handle staff applications to any capacity you choose, from simply forwarding on applications to filling empty spots on your team for you.

PLUGINS

We have a number of specialist plugins available to our member servers. Or, if you need something specific, our group developers will be happy to work with you to come up with something.

COMMUNITY

TeemL is a space for hardworking, dedicated server owners and staff to come together, both to help with server work and to enjoy some time off-duty. We run regular events for all server staff.

ADVERTISING

We're currently working on plans for shared advertising - the details of this are still being confirmed, but watch this space for more info!

HOSTING

We can host websites, bots, APIs and other small things, for both personal and server use, on our servers.
